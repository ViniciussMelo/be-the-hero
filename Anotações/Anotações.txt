Tipos de parâmetros:
- Query Params: Parâmetros nomeados enviados na rota após "?" (Filtros, paginação)
Ex: http://localhost:3333/users?page2&name=Vinicius&idade=19 -> Pagina 2, nome Vinicius e idade 19
request.query

- Route Params: Parâmetros utilizados para identificar recursos
Ex: http://localhost:3333/users/1 -> usuário com Id = 1
request.params 

- Request Body: Corpo da requisição, utilizado para criar ou alterar recursos
request.body

- Request body = Cabeçalho da requisição, caracteriza o contexto da requisição
request.headers

./ -> indicar que é um arquivo e não um pacote.
